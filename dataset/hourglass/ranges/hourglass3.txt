#   b: 2+,3,4-

#
# K3_mid {d, e, h}: alive at lines 5-6
#   d: already defined (alive 2-8, spans left and mid)

#   h: 5+,6,7,8,9,10,11-  (AP2 — bridges mid and right cliques)
#
# K3_right {h, i, j}: alive at lines 9-10

# Interference edges:
#   Left K3:  a-b, a-d, b-d
#   Mid K3:   d-e, d-h, e-h
#   Right K3: h-i, h-j, i-j
#   d and h are articulation points in the block-cut tree.
#
# Chromatic number: 3
# Block-cut tree: leaf(K3_left) --d-- (K3_mid) --h-- leaf(K3_right)

#   For K3_left, a,b,d → any 3-coloring works.

#   reusing colors across cliques (left uses {0,1,2}, mid uses {0,1,2},
#   right uses {0,1,2} — color normalization collapses to 3 registers).
