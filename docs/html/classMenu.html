Detailed Description

Terminal UI renderer — draws boxes, arrow menus, and prompts.

Pure presentation layer. Knows nothing about domain data. Receives pre-formatted strings and renders them using ANSI escape codes.

Interactive filesystem browser starting at startDir.

Directories are listed before files. Selecting a directory descends into it; selecting a file returns its full path. Auto-descends past intermediate subdirectories named hideDirName when entering a folder.
