Detailed Description

Writes the allocation result to a text file in the required format.

Output format (success):

# Total number of webs ...
webs: N
web0: 1+,2,3-
...
registers: K
r0: web0
r0: web2
r1: web1

Output format (failure — all to memory):

webs: N
web0: ...
registers: 0
M: web0
M: web1
