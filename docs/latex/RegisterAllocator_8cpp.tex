\doxysubsection{Detailed Description}
Implementation of the four register allocation algorithms. 

Algorithms implemented:\+
\begin{DoxyItemize}
\item {\bfseries{basic}}:\+ Chaitin-\/style greedy simplification + stack-\/based coloring.
\item {\bfseries{spilling}}:\+ Basic coloring extended with up to K webs committed to memory.
\item {\bfseries{splitting}}:\+ Iterative web splitting to reduce interference, then basic coloring.
\item {\bfseries{free}}:\+ DSATUR heuristic — always color the highest-\/saturation web first. 
\end{DoxyItemize}
