\doxysubsection{Detailed Description}
Converts domain data into display-\/ready string lines for the \doxylink{classMenu}{Menu}. 

Pure formatting layer — receives domain objects, returns vectors of strings. Knows nothing about terminal rendering or ANSI codes.
