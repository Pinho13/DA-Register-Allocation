\doxysubsection{Detailed Description}
Builds and owns the interference graph for a set of webs. 

Each node in the graph corresponds to one web (identified by its id). An undirected edge connects two nodes if their webs interfere.

Two webs interfere if they are simultaneously alive at any program point, with one exception:\+ if web A ends at line N with \textquotesingle{}-\/\textquotesingle{} (last-\/use) and web B starts at line N with \textquotesingle{}+\textquotesingle{} (definition), they do NOT interfere at that line.
