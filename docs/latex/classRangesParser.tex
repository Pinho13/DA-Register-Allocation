\doxysubsection{Detailed Description}
Parses the live-\/ranges input file into a list of \doxylink{structLiveRange}{Live\+Range} objects. 

File format (lines starting with \textquotesingle{}\#\textquotesingle{} are ignored):\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{varName:\ line+,\ line,\ line-\/}
\DoxyCodeLine{varName:\ line+,\ line-\/}

\end{DoxyCode}


A \textquotesingle{}+\textquotesingle{} suffix on a line number marks a definition point. A \textquotesingle{}-\/\textquotesingle{} suffix marks a last-\/use point.
