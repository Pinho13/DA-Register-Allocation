\doxysubsection{Detailed Description}
Dispatcher for all register allocation algorithms. 

Selects and runs the algorithm specified in \doxylink{structRegisterConfig_ab3ed45ad33bd6079e6b09a9628f7eace}{Register\+Config::\+algorithm}. Implementations are split across:\+
\begin{DoxyItemize}
\item \doxylink{Coloring__Basic_8cpp}{Coloring\+\_\+\+Basic.\+cpp} — basic greedy, spill/\+split candidate selection
\item \doxylink{Coloring__Dsatur_8cpp}{Coloring\+\_\+\+Dsatur.\+cpp} — DSATUR heuristic + exact backtracking
\item \doxylink{Coloring__BCT_8cpp}{Coloring\+\_\+\+BCT.\+cpp} — Phantom (BCT-\/\+Color):\+ biconnected-\/component decomposition
\item \doxylink{Coloring__Utils_8cpp}{Coloring\+\_\+\+Utils.\+cpp} — Kempe swap, colour normalisation, reduction, validator 
\end{DoxyItemize}

Runs the register allocation algorithm specified in {\ttfamily config}. 

Dispatches to basic\+Coloring, dsatur\+Coloring, or partitioned\+Coloring depending on config.\+algorithm ("{}basic"{}, "{}spilling"{}, "{}splitting"{}, "{}dsatur"{}, "{}phantom"{}). Applies a DSATUR floor for phantom:\+ falls back to DSATUR when it produces fewer spills.
