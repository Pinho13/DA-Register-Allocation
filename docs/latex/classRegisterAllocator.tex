\doxysubsection{Detailed Description}
Implements all four register allocation algorithms. 

Algorithms:\+
\begin{DoxyItemize}
\item basic:\+ Greedy graph coloring as described in the project spec.
\item spilling:\+ Greedy coloring with up to K webs spilled to memory.
\item splitting:\+ Greedy coloring with up to K webs split into two sub-\/webs.
\item free:\+ DSATUR heuristic (color node with most constraints first). 
\end{DoxyItemize}
