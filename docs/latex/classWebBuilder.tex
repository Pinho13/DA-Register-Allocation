\doxysubsection{Detailed Description}
Merges live ranges into webs using a greedy union approach. 

Two live ranges of the same variable are merged into one web if:\+
\begin{DoxyEnumerate}
\item They share at least one program line number, OR
\item One ends at line N with \textquotesingle{}-\/\textquotesingle{} and the other starts at line N with \textquotesingle{}+\textquotesingle{} (same-\/line def-\/use fusion).
\end{DoxyEnumerate}

Merging is repeated until no further merges are possible (fixed-\/point).
