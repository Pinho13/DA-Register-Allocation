\doxysubsection{Detailed Description}
Represents a single live range for a variable:\+ a set of program line numbers. 

A line number may carry a \textquotesingle{}+\textquotesingle{} (definition point) or \textquotesingle{}-\/\textquotesingle{} (last-\/use point) marker. These are stored separately so interference logic can use them.
